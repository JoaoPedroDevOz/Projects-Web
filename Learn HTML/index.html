Modelando caixas

padding: espaçamento entre o conteúdo e a borda da caixa

margin: comprimento da borda da caixa

outline: contorno da borda

Caixa box-leve:

caixa que ocupa uma linha inteira da página, e depois quebra uma linha.
EX: div, h1-h6, p, main, header, nav, article, aside, footer, form, video

Caixa inline-level

caixa que ocupa apenas um espaço da linha na página, também não quebra linhas.
EX: span, a, code, small, strong, em, sup-sub, label, button, input, select

Sombras nas caixas

para colocar sombra na caixa usa o código "box-shadow": (sombra para as horizontais) (sombra para as verticais) (espessura, blur, opacidade da sombra) (cor da sombra). Ex: box-shadow: 6px 15px 7px black

Caixas com vértices arredondados

para arredondar a caixa usa o código "border-radius"

Bordas decoradas

a borda decorada é uma imagem que é usada no código border-image-source: url()

Bordas other commands

border-image-slice, serve para fatiamento da borda
border-image-repeat: repeat, serve para repetição da borda.
